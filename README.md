# 里山 植物調査

約1ヘクタールの土地（耕作放棄地）の植物生態を、iPhoneで撮影・位置・名前を記録するためのWebアプリです。  
里山開拓を総合支援するスマートフォンアプリの**第一段階**として利用できます。

## できること

- **写真**: スマートフォンのカメラで植物を撮影（タップで撮影エリアを表示）
- **位置**: 「現在地を取得」でGPSの緯度・経度を記録
- **植物名・メモ**: 名前とメモを入力して保存
- **一覧**: 記録を一覧表示
- **エクスポート**: 全記録をJSONでダウンロード（バックアップ・他ツール連携用）

データはブラウザのIndexedDBに保存されるため、**同じ端末・同じブラウザ**で継続して使う必要があります。定期的に「JSONでエクスポート」してバックアップすることをおすすめします。

## 使い方

### 1. アプリを開く

- **開発中**: プロジェクトの `plant-survey` フォルダで簡易サーバーを起動し、iPhoneのブラウザでアクセスします。

  ```bash
  cd plant-survey
  npx serve -l 3000
  ```

  または Python がある場合:

  ```bash
  cd plant-survey
  python3 -m http.server 3000
  ```

  同じWi‑Fi内のiPhoneのSafariで `http://あなたのPCのIP:3000` を開くか、  
  PCのブラウザで `http://localhost:3000` を開いて動作確認できます。

- **本番想定**: 後からHTTPSのサーバー（例: GitHub Pages、Netlify、自前サーバー）に置くと、どこからでもiPhoneで開けます。  
  **位置情報を使う場合はHTTPS（または localhost）が必要**です。

### 2. 記録の流れ

1. 「タップして写真を撮影」をタップ → カメラが起動するので植物を撮影
2. 「現在地を取得」をタップ → 位置情報の許可を出されたら「許可」
3. 「植物名」を入力（例: ススキ、クズ、ヨモギ）
4. 必要なら「メモ」に生育状況・量・高さなどを入力
5. 「記録を保存」をタップ

記録は下の「記録一覧」に追加され、一覧で確認できます。

### 3. エクスポート（バックアップ）

「JSONでエクスポート」をタップすると、全記録（写真はBase64で含む）が入ったJSONファイルがダウンロードされます。  
ファイル名は `satoyama-plant-survey-YYYY-MM-DD.json` の形式です。PCにコピーして保管したり、後のアプリで読み込むのに使えます。

## 技術構成

- 単一HTML + CSS + JavaScript（ビルド不要）
- IndexedDB でローカル保存
- Geolocation API で位置取得
- モバイル対応（viewport・タッチ・safe-area）

## 今後の拡張案

- 植物の同定API連携（撮影から名前の候補を表示）
- 地図上に記録ポイントを表示
- 分類・タグ（草本・木本、在来・外来など）
- クラウド同期・複数デバイス対応
